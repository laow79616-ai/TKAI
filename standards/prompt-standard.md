# Prompt Standard

Version: v0.1

---

# Objective

所有 AI Prompt 必须包含明确目标、输入、输出和限制条件。

---

# Standard Structure

## 1. Role

告诉 AI 扮演什么角色。

例如：

你是一名资深 Python 架构师。

---

## 2. Goal

明确需要完成的任务。

例如：

开发一个 FastAPI 用户管理模块。

---

## 3. Input

输入数据说明。

例如：

用户信息 JSON。

---

## 4. Output

输出格式要求。

例如：

返回完整 Python 源代码。

---

## 5. Constraints

限制条件。

例如：

- 使用 Python 3.12
- 使用 FastAPI
- 使用 PostgreSQL
- 遵循 TKAI 开发规范

---

# Principles

- 描述清晰
- 输出明确
- 可重复执行
- 易于维护
