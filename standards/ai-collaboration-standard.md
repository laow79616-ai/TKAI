# AI Collaboration Standard

Version: v0.1

---

# Objective

建立统一的 AI 协作流程，提高开发效率，降低沟通成本。

---

# AI Roles

## ChatGPT

负责：

- 系统架构设计
- 产品规划
- 技术方案
- 文档编写
- Code Review
- 开发计划

---

## Claude Code

负责：

- 编写代码
- 修改代码
- 重构代码
- 修复 Bug
- 自动生成项目

---

## Cursor / Copilot

负责：

- 代码补全
- 局部修改
- 快速开发
- 单元测试

---

# Development Workflow

1. ChatGPT 输出开发方案
2. Claude Code 生成代码
3. Cursor 辅助开发
4. ChatGPT Review
5. Git Commit
6. Push GitHub

---

# Principles

- 文档先行
- 规范统一
- 小步提交
- 自动测试
- 持续优化
