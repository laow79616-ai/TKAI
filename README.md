# TKAI

> AI Software Factory

## Overview

TKAI（Toolkit AI）是一个 AI 软件工厂，用于统一管理 AI 软件开发流程。

本项目不负责具体业务，而是负责：

- 项目模板（Templates）
- AI Prompt 库（Prompts）
- 开发规范（Standards）
- 产品文档（Docs）
- Sprint 与任务管理（Tasks）
- AI 协作规则（CLAUDE.md）
